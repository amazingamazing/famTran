# Family Readiness Checklist

Use this checklist during the 2 weeks before your family arrives.

## Accounts and keys

- [ ] GitHub repo created and private access configured.
- [ ] Render service deployed from `render.yaml`.
- [ ] `DEEPGRAM_API_KEY` set in Render environment.
- [ ] `GEMINI_API_KEY` set in Render environment.
- [ ] `CARTESIA_API_KEY` set in Render environment.

## Device readiness

- [ ] Two iPhones tested with PWA installed from home screen.
- [ ] Auto-lock disabled during conversation runs.
- [ ] Earbuds paired and verified on each test device.

## Rehearsal sessions

- [ ] Run at least 5 sessions of 10-15 minutes with your wife.
- [ ] Log mistranslations and immediately submit corrections from UI.
- [ ] Add glossary entries for every repeated family-specific term.
- [ ] Validate reconnect behavior by backgrounding and resuming app mid-session.

## Final week freeze

- [ ] Pick the best provider combination from your tests.
- [ ] Stop feature changes 7 days before travel.
- [ ] Keep only bugfixes during freeze week.
- [ ] Run `npm run test && npm run build` before final deploy.
